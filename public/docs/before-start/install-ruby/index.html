    Install Python #

    How to install Python #

    There a couple of ways to install Python in your computer:

    • You can go to the official page and follow the given instructions for your operation system.
    • You can use the Python Version Manager RVM.

    Validate Python is installed #

    Once you have Python installed you should be able to run the following command in the terminal:

    python -v # => python 3.1.2p20 (2022-04-12 revision 4491bb740a) [x86_64-linux]

    Also you should be able to open an interactive python shell using the following command in the terminal:

    irb

    In which you can run Python code like this one:

    irb(main):001> 1 + 1 # => 2
